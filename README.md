<h2 align="center">⚙️ Setup Instructions</h2>

<p align="center">
Follow the steps below to configure the SSH Monitoring Dashboard in <b>Splunk</b>.
</p>

---

### 📌 Step 1 — Add Dashboard XML
Copy the contents of the `dashboard.xml` file.

Then open the Dashboard Source Editor and paste the XML code to create the dashboard.

---

### 📂 Step 2 — Upload Sample Logs
Upload the `ssh_logs_new.json` file to add the sample SSH log data.

---

### ⏰ Step 3 — Set Time Range
Set the dashboard time range to: All Time
