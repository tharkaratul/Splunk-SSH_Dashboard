## Setup Instructions

1. Copy the contents of the `dashboard.xml` file.
2. In Splunk, open the dashboard source editor and paste the XML code to create the dashboard.
3. Upload the `ssh_logs_new.json` file to Splunk to add the sample SSH log data.
4. Set the dashboard time range to **All Time** to ensure all uploaded log data is displayed correctly.
5. Once the data is available in Splunk, the dashboard visualizations and alerts will populate automatically.
